🆘 حل المشاكل الشائعة:

  ❌ ModuleNotFoundError
     → pip install -r requirements.txt

  ❌ Port 5000 already in use
     → python run.py --port 5001

  ❌ Database error
     → احذف worker_management.db وأعد التشغيل
